# Where things live — grep anchors per module

One line per module: key exports / methods / knob sites by *name* (grep these;
line numbers rot, names don't). Architecture & gotchas stay in CLAUDE.md.

## src/

- `geo.js` — `GEO` singleton, `hAt` (terrain height, `ELEV` grid consts), `nearestRoad` (spatial grid, optional type filter), `nearestCity`, `inTexas`, `countyAt`, `waterAt`, `seededRand` (never change seed strings).
- `world.js` — `buildWorld` → `buildTerrain`/`buildHighways`/`buildWater`/`buildCountyLines`/`buildRibbons` (all roads = merged meshes per tier); `ScenerySystem` inside `buildWorld` return (chunked flora/props, `group.userData.animated`); `chapelAt`/`chapelSitesNear` (pure seeded chapel+cemetery sites, `mkChapel`/`mkCemetery`).
- `vehicle.js` — `Player`: `update` (physics branches; DRIVE `caps{}`, offroad/rain via `this.perks`), `animate` (lights/particles; headlight intensity/lead knobs), `setMode`, `resetToRoad`; `mkTruck` (`userData`: headlights/wheels/brakes/cargo/beams/bodyMat), `mkWings` (`userData`: mat/stockColor — missions.js charter livery swap), `mkCowboy`, `mkStarMesh`.
- `sky.js` — `ATMOS` (mutable: wind/night/weather/rain/ufo), `SkySystem.update` (weather state machine: `nextPick`, `forecast`/`forecastT` radio window, `blend`; time keyframes `KEYS`), `WEATHER`/`ODDS` tables, `forecastLine`, `weatherIcon`, `skyReport`, `buildCelestial`, `DAY_SECONDS`.
- `gameplay.js` — `Gameplay`: save load/`persist` (keys: cities/landmarks/roses/species/stats/counties/ufo/bank/jobsDone/job/gear/legends/airports), `LANDMARKS` (+`LL` projection), `enterCounty`, `spotSpecies`, `spotLegend`, `ufoSighting`, `logAirport` (10th collectible, dedup by field id), `landmarkNear`, `mkRoses` (rose RNG — never touch).
- `missions.js` — `MissionSystem`: `genOffers` combines `genGroundOffers`/`genCharterOffers` (pay/deadline knobs at top, `CARGO`/`MANIFEST`/`REAL_ROUTES` tables), `accept`/`load`/`deliver`/`finishJob` (payout multipliers, kind-branched), `abandon`, `hudInfo`, `target`/`field` (city vs. airport waypoint), `setLivery`, `force(fromId,toId)` charter test hook, guide arrow in `update`/`toggleArrow`.
- `shop.js` — `SHOP` catalog, knob arrays (`ENGINE_CAP`/`TIRE_*`/`LIGHT_I` — index 0 = stock, match vehicle.js), `PAINTS`/`PAINT_PRICE`, `buy`/`buyPaint`/`gearLevel`, `applyGear` (→ `player.perks` + bodyMat + dog; `perks.avionics` = aviation-band radio).
- `dog.js` — `DogSystem`: `update` (bed perch vs WALK follow; `FOLLOW_D`/`MAX_SPD` knobs), `honked` (delayed yips), `setOwned`, `mkDog`.
- `travel.js` — `TravelMenu`: `render` (tab dispatch), `renderJobs`, `renderShop`, `buyItem`, `jobClick`, `go` (arrival modes), `NATURE`/`ICONS` POI tables.
- `hud.js` — `HUD`: `update` (12 Hz; location/road/speed/mode/forecast lines), `renderMapLayer` (one offscreen canvas), `drawMini`/`drawBig`/`drawCompass`, `uiScale` (rem root), `toast`/`dialog`/`interactHint`/`subtitle(text, header)` (radio, one-line queue, `#radio-header`/`#radio-text`), `updateTags(sources, camera)` + `tagLabel`/`tagPool` (A5 aircraft tags over `radio.sources`), `toggleBigMap`/`toggleHelp`/`cycleZoom`.
- `audio.js` — `AudioSystem`: `update` (engine/wind/rain/crickets/theremin), `chime` (`SONGS` table), `radio(text, opts)` (tower-radio synth; `radioDuck` gain node, `opts.ufo` chop, `opts.voice` {p,r} per-type pitch/rate), `heli(dist)` (rotor ambience, 12 Hz-chopped bandpass noise; `heliTarget` — the last commanded gain, for tests), `honk`/`bark`/`howl`/`rattle`/`gobble`/`trainHorn`/`thunder`/`flare`/`step`, `toggleMute`.
- `traffic.js` — `TrafficSystem`: `update` (follow/brake/honk/pull-around; `DENSITY_DIVISOR` etc. at top), `refreshCandidates` (supply-based density), `junctionHop`, `spawn`/`pickType`/`mixAt`, vehicle builders `mkSedan`/`mkPickup`/`mkSuv`/`mkSemi`.
- `animals.js` — `SPECIES` table (speeds/behaviors/night gates/facts), `regionTable` (boxes mirror world.js), `AnimalSystem`: `update`/`flee` (heading: away = atan2(-dx,-dz))/`scare`/`sound`, `mkAnimal`.
- `npcs.js` — `NPCSystem`: `interact` (dialog assembly; B2 heli/job/forecast context pulls), `startle`, `spawnTownsfolk`, `spawnBystanders`/`bystanderConvo` (B1 gate figures, `gatefolk:` stream, `nextSlot` schedule scan, `this.aviation` assigned by main.js), `roadShoulder`, `mkCharacter`, `POOLS` (all content pools, verify surface; `PROFESSION_LINES` flavored smalltalk, `BYSTANDER_ROLE_INFO` variant age bands).
- `cities.js` — `CitySystem`: `spawn` (InstancedMesh buildings, `hasRealStreets`), `setNight` (window glow), `cityRadius` (visit radii).
- `airports.js` — `AIRPORTS` table (true headings from OSM; `RW_W`/`MARGIN` tier knobs; pure per-site `anchor`/`gate` computed at module load), `airportClear` (pure footprint exclusion — called by cities.js/ScenerySystem/`chapelAt`), `onRunway` (pure pavement-corridor test; `clearOfRunways` is its negation), `TD_AGL`/`TD_SPD` (touchdown thresholds — the shared definition of "landed", read by radio.js and missions.js), `airportLayout` (pure, pads at max `hAt`; `padYOf` single-site scan), `padAt(id)` (pure helipad spot beside the anchor — A4 medical pad stops land here), `windFrom` (per-day seeded wind direction), `runwayInUse` (pure argmax end into `windFrom` — AI ops + radio.js ATIS share it), `rwyLabel` (spoken runway number — ATIS + B1 gate spotter), `AirportSystem` (8 merged meshes in `this.group`; beacon spin + windsock droop in `update`).
- `aviation.js` — `daySchedule` (pure seeded departures; `SLOTS`/`REDEYE_MAX`/night-u window consts), `AviationSystem`: `buildRaw` (closed-form trajectory legs; speed/altitude knobs in `TYPES`/`SLOPE`/`CEIL`), `evalFlight`, `update` (materialize scan within `MAT_R`, `MAX_AIR` cap, storm/dust ground stop + `divert` go-around, dt-vs-days delay model), `divert(m)` (public go-around trigger, reused by radio.js), `force('departure'|'arrival')`, `despawnAll`, `ROUTES` (real weighted pairs), `mkJet`/`mkGa`, `simT` (real-loop sentinel).
- `radio.js` — `TowerRadio`: `receivable` → `{a, kind: 'tower'|'unicom'}` (FLY-in-range — 250u tower/120u UNICOM — or `perks.avionics`), `update(dt, player, aviation, sky)`, `atis`/`awos`/`narrateOps`/`playerFlow` (tower only)/`checkBlock` (physical go-around, reception-independent, all 20 fields), `tx`/`lastTx` (structured, for tests; A3 adds `{at, src, cs, route, phase, casual, voice, header}`), A3 chatter: `scan` (→ `this.sources`, `DIRECT` 60u window — one enumeration for chatter + hud tags), `chatter` (`EDGE_EV` phase edges, `ROLL_OK` enroute rolls, `GAP_MIN`/`GAP_VAR` 25–45 s budget, `OPS_HOLD` preempt, `playerRefT` throttle + `REF_SPD` gate), `ctxFor` (live template context), `helis`/`militaryAir` (assigned by main.js), `TOWERED`/`TOWERED_COUNT` (7 tier-1 hubs), `UNICOM` (9 tier-2 fields), `nearestTowered`/`nearestUnicom`.
- `chatter.js` — A3 line pools, pure (geo.js only): `VOICES` ({p,r} synth character per type, consumed by audio.radio), `HELI_ID` (kind → callsign/operator/voice: Lifeguard 3/StarCare, Chopper 5/KTX News 5, Rescue 6-0, Hood 2-1), `POOLS` (kind×event templates, `{token}`-gated = factual by construction), `AIRLINE_POOLS` (per-carrier registers), `chatterLine(kind, event, ctx, seed)` (seeded `chatter:` pick, ~1-in-4 humor rationing).
- `rotors.js` — `HeliSystem` (medical/news/coastguard/army candidates — each exposes `ph`/`city`; medical adds `fieldId`/`pad`/`sortieN`, `force(kind, {pad})`/`despawnAll`/`airborneCount`/`nearestAirborneDist`, `startSortie` (seeded `padstop:${city}:${day}:${sortie}` roll, `PAD_ODDS`/`MED_SPD`/`PAD_DESC_T` knobs), `advancePadSortie` (transit→descend→pad→lift→return timeline), global `CAP=2` incl. weighted Army pair, `update(dt,px,pz,days)` real-loop `simT`), `BlimpSystem` (`positionAt(day,u)` pure per-day loiter query, `loiterAnchor`/`loiterParams`, `update(dt,days)` loiter/transit/moored state machine, `simT`).
- `military.js` — `MilitaryAirSystem` (wave 5): `candidates` ('nasa' rare arrival into Ellington, 'lowlevel' rare fast pass gated to the Trans-Pecos box `x < -2200`), `force(kind, aviation)`/`despawnAll`/`airborneCount`/`nearestAirborneDist`, shares `aviation.js`'s `MAX_AIR` fixed-wing cap via `aviation.airborneCount()` (own weight `PAIR_W=2`), `update(dt,px,pz,aviation)` real-loop `simT`.
- `flares.js` — `FlareSystem`: `fire`/`snuff`/`update` (ballistic→chute phases; `LIGHT_I`/`BURN`/`CHUTE_FALL`/recharge knobs at top).
- `trains.js` / `maritime.js` / `ufo.js` / `bats.js` — `TrainSystem` (`arcInit`/`at` arc-length follow, `mkLoco`), `MaritimeSystem` (`laneAt`, `buildPorts`/`buildPlatforms`/`buildShips`), `UFOSystem` (`hotspotBoost`, `startSaucer(px,pz,immediate)`/`startFormation`; hover *stalks* the player — standoff/height/orbit knobs in the `tgt` block, `near` ramp drives the Levelland flicker), `BatSystem` (dusk window on `sky.t`).
- `haunts.js` — `LEGENDS`/`LEGEND_COUNT`/`EROCK`, `HauntSystem.update` (wisp arm/fade, ghost fires, midnight bell via `lastBell`, debug `force` flag; gates/radii knobs at top: `WISP_ODDS`/`NIGHT_MIN`/`FADE_*`/`WATCH_*`/`BELL_R`); chapel sites come from world.js `chapelAt`/`chapelSitesNear`.
- `debug.js` — `initDebug` playtest menu (`?debug=1` + backquote): `actions` table (day/night/midnight, `hauntCemetery`, `ghostFires`, `saucer`/`formation`, `bats`, weather picks) always on `__game.debug`; only the panel is URL-gated.
- `main.js` — `boot`: system construction order, keydown map (horn/interact/travel/…), render loop (update call order, 12 Hz `hudTick` block, `__skipRender` gate), `window.__game` exposure.

## tools/

- `verify.mjs` — harness: `check`, `t.ev/tp/wait/simWait/simStep/step/hold/release/key/until/setTime/setDay/setNight/setWeather/sample/shot/stubGamepad`; compact output default, `-v` verbose; `__skipRender` set at boot.
- `checks/*.mjs` — suites: debug/drive/haunts/hud/lights/missions/shop/traffic/wildlife (one default-export async fn each; debug runs first alphabetically and must restore daylight/clear weather).
- `build-data.mjs` (`proj` — duplicated as `LL` in gameplay.js/travel.js), `build-elevation.mjs` (grid consts ↔ `ELEV`), `build-sky.mjs`, `add-metro-streets.mjs` (append-only), `status.sh`.
